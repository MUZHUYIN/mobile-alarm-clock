# 自由设定时长闹钟 - 移动应用版

这是一个基于PWA (Progressive Web App) 技术的移动闹钟应用，可以在手机上安装并离线使用。

## 功能特点

- 自由设定倒计时时长（分钟和秒）
- 直观的倒计时显示
- 开始、暂停、重置功能
- 多种铃声选择
- 深色/浅色主题切换
- 倒计时结束后自动响铃提示
- 响铃后自动重置并开始新的倒计时
- 支持离线使用
- 可安装到主屏幕

## 如何使用

### 方法一：直接访问网页版

1. 在手机浏览器中访问部署好的应用URL
2. 点击浏览器菜单中的"添加到主屏幕"选项
3. 应用将被安装到手机主屏幕，可像原生应用一样使用

### 方法二：本地部署

1. 克隆或下载本项目到本地服务器
2. 使用HTTP服务器（如Apache、Nginx）部署项目
3. 在手机浏览器中访问部署地址
4. 添加到主屏幕

## 技术实现

- 基于HTML5、CSS3和JavaScript开发
- 使用Tailwind CSS进行样式设计
- 使用GSAP实现动画效果
- 使用PWA技术实现移动应用功能：
  - Web App Manifest 配置应用信息
  - Service Worker 实现离线缓存和推送通知

## 浏览器兼容性

- Chrome 67+
- Firefox 60+
- Safari 12.1+
- Edge 79+

## 注意事项

1. 首次使用需要联网以缓存必要资源
2. 为确保闹钟功能正常工作，请不要关闭浏览器通知权限
3. 在某些浏览器中，自动播放音频可能受到限制，需要用户交互后才能播放
